# Retail-AI-assistant-using-SLM-with-PPO-and-VERL
Project for Reinforcement Learning in IIIT Dharwad

This AI assistant represents a fundamental evolution in retail customer service by combining three cutting-edge technologies:
Core Capabilities:
1.  Intelligent Conversational Commerce - Engages in natural, multi-turn conversations with customers
2.  Personalized Product Recommendations - Dynamically suggests products based on real-time customer profiling
3.  Autonomous Decision Making - Learns optimal conversation strategies through reinforcement learning
4  Safety-Critical Operations - Continuously monitors and verifies its own behavior for safety and fairness
